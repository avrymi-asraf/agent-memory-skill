# Agent Memory Skill

This repository is a project-level template for agent files, skills, memory, and MCP configuration.

The usable starter lives in `project-template/`. Copy its contents into a project root. Do not install these files globally and do not symlink them from dotfiles; the system is intentionally project-scoped so each repository owns its agents, skills, memory, and MCP settings.

See `agent-memroy-skill.md` for the design and the OpenCode / GitHub Copilot template map.
