# Internet Search

Use current sources when information may have changed, when recommendations affect money or time, or when the user asks to verify.

## Source Priority

- Official docs for technical APIs.
- Primary sources for laws, releases, pricing, standards, and products.
- Recent reputable sources for news.

## Output

Separate facts from inference. Include source links. Use exact dates when relative timing could confuse the answer.

## Project Knowledge

If research creates durable project knowledge, update a project skill or wiki note after answering.
